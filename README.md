Molecular Docking Tutorials

AutoDock Vina • AutoDock4 • AutoDock CrankPep

Repositório contendo tutoriais passo a passo para docking molecular aplicado à bioinformática estrutural,
utilizando ferramentas amplamente empregadas em descoberta de fármacos e interação proteína–ligante.
O objetivo é fornecer workflows reproduzíveis, desde a preparação das estruturas até a análise dos resultados.

Sobre o Projeto

Este repositório apresenta guias práticos para execução de docking molecular utilizando diferentes abordagens:

⚡ Docking rápido e eficiente (AutoDock Vina)

🔬 Docking clássico com funções de energia detalhadas (AutoDock4)

🧬 Docking proteína–peptídeo (AutoDock CrankPep)

📖 Público-Alvo

• Bioinformatas

• Químicos computacionais

• Pesquisadores em descoberta de fármacos

• Estudantes de biologia estrutural

• Pesquisadores trabalhando com peptídeos bioativos
