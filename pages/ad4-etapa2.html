Segunda etapa: Preparo do receptor

Abra o AutoDockTools-1.5.7, adicione o receptor, remova a água, remova o ligante complexado, adicione hidrogênios polares, adicione cargas Kollman e salve o receptor em .pdbqt.

1 Remover água
File → Read Molecule
Edit → Delete Water
Vídeo: Remoção de água
2 Remover ligante complexado
Select → Select From String
Residue Sets → Ligand → Add → Dismiss
Edit → Delete → Delete Selected Atoms → Continue
Vídeo: Remoção do ligante complexado
3 Adicionar Hidrogênios Polares
Edit → Hidrogens → Add → Polar Only → OK
Vídeo: Adição de hidrogênios polares
4 Adicionar Cargas
Edit → Charges → Add Kollman Charges
Vídeo: Adição de cargas Kollman
5 Salvar receptor em .pdbqt
Grid → Macromolecule → Choose → selecione seu receptor
Select Molecule → OK → salvar como 6WX4.pdbqt
