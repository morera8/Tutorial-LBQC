Terceira etapa: Grid box

1 Selecione os resíduos que compõem o sítio

Na barra lateral à esquerda (guia Dashboard) clique no mais (+) ao lado do receptor para exibir os resíduos. Selecione os resíduos do sítio de ligação clicando no círculo à direita (select/deselect (S)).

Vídeo: Seleção dos resíduos do sítio ativo
🔢 Resíduos utilizados

106, 109, 110, 111, 162, 163, 164, 247, 248, 264, 268, 271, 272, 286

2 Determinar tamanho da caixa e centro da caixa
Grid → Grid Box → File → Close saving current
💡 Dica

Em View → Show box as lines facilita a exibição da caixa

Vídeo: Definindo o Grid Box
3 Salvar arquivo GPF
Grid → Output → Save GPF → salve como grid.gpf
📁 Localização

Salve na pasta Dock4

Vídeo: Salvando o arquivo GPF
