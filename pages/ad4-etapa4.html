Quarta etapa: Criar DPF

Passos para criação do arquivo DPF:

1. Docking → Ligand → Open → selecione um ligante → Open → Accept
2. Docking → Macromolecule → Set Rigid Filename → escolha o receptor → Open
3. Docking → Macromolecule → Set Flexible Residues Filename → escolha o ligante → Open
4. Docking → Ligand → Choose → Selecione o ligante → Select ligand → Accept
5. Docking → Search Parameters → Genetic Algorithm (padrão) → Accept
6. Docking → Docking Parameters → Accept
7. Docking → Other Options → AutoDock 4.2 Parameters → Accept
8. Docking → Output → Lamarckian → salve como DPF.dpf
💡 Atenção

Escolha a opção adequada para seu docking (Lamarckian, Genetic Algorithm, etc.)
