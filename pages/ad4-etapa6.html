Sexta etapa: Análise dos resultados

1 Passos para análise
→ Analyze → Dockings → Open → Selecione o .dlg do ligante → Abrir → OK
→ Analyze → Macromolecule → Open → OK → Selecione o receptor.pdbqt → Abrir
→ Analyze → Conformations → Play

Na nova janela clique em & → abrirá a janela Set Play Options → selecione Show Info → Conformation

📊 Resultado

Mostra o resultado de todas as conformações do ligante

2 Critérios para escolha da melhor conformação
Parâmetro O que procurar
binding_energy Quanto mais negativo, melhor
ligand_efficiency Mais próximo de zero, melhor
inhib_constant Mais próximo de zero, melhor
3 Criar complexo proteína-ligante
Set Play Options → Write Complex → renomear.pdbqt → Salvar
