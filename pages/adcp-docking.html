Docking AutoDock CrankPep

1 Preparação da pasta de trabalho

No seu computador crie uma pasta de trabalho (ex: ADCP_dock)

2 Preparo do PDB

No PyMol:

  • Limpe a estrutura PDB (água e ligantes indesejados)
  • Remova o peptídeo da proteína
  • Adicione o arquivo .pdb na pasta ADCP_dock
3 Preparo do docking

No terminal, navegue até ADCP_dock e execute:

# Protonar receptor
reduce 3I5R.pdb > 3I5R_H.pdb

# Converter para .pdbqt
prepare_receptor -r 3I5R_H.pdb
Vídeo: Preparo do receptor para docking
4 Gerar target file (.trg) manualmente

No terminal execute:

agfrgui
1. Em Receptor pdbqt adicione seu receptor protonado
2. Na engrenagem em ligand binding pockets selecione version 1.1
3. Clique em Compute pockets [AutoSite 1.1] e selecione seu sítio
4. Em docking box clique em "box all displayed ligand binding pocket" e ajuste o tamanho com padding
5. Clique em generate target file ... → renomeie (ex: 111.trg) → Save
Vídeo: Criação manual do target file
5 Executar o docking
adcp -t 111.trg -s aqtqslvypfpgpipk -N 20 -n 50000 -o pep1_docking > pep1.log
Argumento Descrição
-t arquivo target
-s sequência do peptídeo
-N número de réplicas (buscas independentes)
-n passos de Monte Carlo por réplica
-o prefixo dos arquivos de saída
-ref estrutura de referência (para RMSD)
