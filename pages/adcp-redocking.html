Redocking com AutoDock CrankPep

1 Preparação da pasta de trabalho

No seu computador crie uma pasta de trabalho (ex: ADCP_red)

2 Preparo do PDB

No PyMol:

  • Limpe a estrutura PDB (água e ligantes indesejados)
  • Separe a proteína do peptídeo
  • Adicione ambos os arquivos .pdb na pasta ADCP_red
3 Preparo do Redocking

No terminal, navegue até ADCP_red e execute:

# Protonar receptor e peptídeo
reduce 3I5R.pdb > 3I5R_H.pdb
reduce peptide.pdb > peptide_H.pdb

# Converter para .pdbqt
prepare_receptor -r 3I5R_H.pdb
prepare_ligand -l peptide_H.pdb

# Gerar target file (.trg)
agfr -r 3I5R_H.pdbqt -l peptide_H.pdbqt -o 3I5R
📦 Target file (.trg)

Contém: estrutura do receptor, mapas de afinidade (AutoGrid4), tipos atômicos suportados, pontos de translação (AutoSite) e metadados do docking.

4 Executar o docking
adcp -t 3I5R.trg -s krplpplps -N 20 -n 1000000 -o 3I5R_redocking -ref peptide_H.pdb
📄 Saída

Retorna:

  • Estruturas finais ranqueadas após clustering (3I5R_redocking_*.pdb)
  • Trajetórias Monte Carlo (3I5R_redocking_*.out)
  • Arquivos principais (3Q47_redocking_ranked_*.pdb)
  • Tabela de resultados no terminal com pose, afinidade (kcal/mol) e RMSD
