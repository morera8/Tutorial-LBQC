Segunda etapa: Preparo do receptor

Abra o AutoDockTools-1.5.7, adicione o receptor, remova a água, remova o ligante complexado do PDB baixado, remova ligantes indesejados, adicione hidrogênios polares e salve o receptor em .pdbqt

Vídeo: Visão geral do preparo do receptor
1 Remover água
File → Read Molecule
Em seguida: Edit → Delete Water
Vídeo: Remoção de água
2 Remover ligante complexado

Na barra de tarefas: Select → Select From String

→ Residue Sets → selecione Ligand → clique em Add → clique em Dismiss
→ Isso seleciona o ligante a ser removido

Na barra de tarefas siga: Edit → Delete → Delete Selected Atoms → Continue

Vídeo: Remoção do ligante complexado
💡 Nota

Deletar átomos do ligante selecionado

3 Adicionar Hidrogênios Polares
Edit → Hidrogens → Add → Polar Only → OK
Vídeo: Adição de hidrogênios polares
ℹ️ Importante

Vina utiliza uma função de pontuação de átomos unidos. Assim como no AutoDock, os hidrogênios polares são necessários nas estruturas de entrada para classificar corretamente os átomos pesados como doadores de ligações de hidrogênio (Manual - AutoDock Vina, acesso 17/01/2026).

4 Salvar receptor em .pdbqt
Grid → Macromolecule → Choose → selecione seu receptor → Select Molecule → OK → salvar receptor como 6WX4.pdbqt
📁 Localização

Salve na pasta Dock_Vina

Vídeo: Salvando o receptor em formato .pdbqt
