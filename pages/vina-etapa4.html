Quarta etapa: Preparo do ligante

1 Converter ligante para .pdb

Na guia Dashboard oculte o receptor clicando no círculo à direita (display/undisplay (L)).

No PyMol:

File → Open → selecione o arquivo .sdf → Abrir
File → Export Structure → Export Molecule → geranyl_isobutanoate.pdb → Salvar
💡 Atenção

Antes de salvar mude o tipo de arquivo para .pdb

Vídeo: Conversão do ligante para .pdb
2 Adicionar Hidrogênios Polares

No AutoDockTools:

Ligand → Input → Open → geranyl_isobutanoate.pdb → Open
Edit → Hidrogens → Add → Polar Only → OK
Vídeo: Adição de hidrogênios polares ao ligante
3 Salvar ligante em .pdbqt
Ligand → Output → Save as PDBQT → salvar ligante como geranyl_isobutanoate.pdbqt
📁 Localização

Salve na pasta Dock_Vina

Vídeo: Salvando o ligante em .pdbqt
