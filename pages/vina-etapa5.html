Quinta etapa: Execução do Docking

1 Windows

Execute o comando:

vina_1.2.7_win.exe --config conf.txt
📄 Resultado

O comando irá gerar o arquivo res_out.pdbqt contendo todas as conformações e suas respectivas energias.

Para separar cada conformação em um arquivo use o comando:

vina_split_1.2.7_win.exe --input res_out.pdbqt
Vídeo: Execução do docking no Windows
2 Linux

Execute os comandos:

chmod +x ./vina_1.2.7_linux_x86_64
./vina_1.2.7_linux_x86_64 --config conf.txt
📄 Resultado

O comando irá gerar o arquivo res_out.pdbqt contendo todas as conformações e suas respectivas energias.

Para separar cada conformação em um arquivo use os comandos:

chmod +x ./vina_split_1.2.7_linux_x86_64
./vina_split_1.2.7_linux_x86_64 --input res_out.pdbqt
