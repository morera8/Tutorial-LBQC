Múltiplos Ligantes

1 Primeira etapa: Preparação da pasta de trabalho

Crie uma pasta de trabalho contendo:

  • receptor.pdb
  • pasta ligantes contendo ligantes a serem convertidos
  • pasta resultados
  • pasta ligantes_conv para salvar os ligantes convertidos
  • arquivo conf.txt sem ligand e out
  • vina_1.2.7, vina_split_1.2.7 e script multiplo_lig.py
📁 Nome da pasta

Aqui o nome da pasta será Mult_lig

2 Segunda etapa: Preparo do receptor

Processo idêntico ao preparo do receptor para docking único no AutoDock Vina.

1. File → Read Molecule
2. Edit → Delete Water
3. Select → Select From String → Residue Sets → Ligand → Add → Dismiss
4. Edit → Delete → Delete Selected Atoms → Continue
5. Edit → Hidrogens → Add → Polar Only → OK
6. Grid → Macromolecule → Choose → Select Molecule → OK → salvar como 6WX4.pdbqt
📁 Localização

Salve na pasta Mult_lig

Vídeo: Preparo do receptor para múltiplos ligantes
3 Terceira etapa: Grid box

Processo idêntico à definição do Grid Box para docking único.

1. Selecione os resíduos que compõem o sítio
2. Determine tamanho da caixa e centro da caixa
Vídeo: Definição do Grid Box para múltiplos ligantes
4 Quarta etapa: Arquivo conf.txt

Remova as linhas ligand e out:

receptor = 6WX4.pdbqt

center_x = 11.938
center_y = -25.901
center_z = -37.072

size_x = 56
size_y = 72
size_z = 70

exhaustiveness = 8
📁 Localização

Salve na pasta de trabalho Mult_lig

5 Quinta etapa: Preparo dos ligantes

No Open Babel:

1. INPUT FORMAT → selecione o formato de entrada dos seus ligantes
2. Clique em "..." → procure a pasta com todos os ligantes a serem convertidos
3. Abaixo de CONVERT selecione: "Adds hydrogen to polar atoms only"
4. OUTPUT FORMAT → selecione o formato de saída (.pdbqt)
5. Clique em "..." → selecione a pasta de destino dos ligantes convertidos
💡 Renomeação

Renomeie desta forma (*.pdbqt) para preservar os nomes. Exemplo:

~\Dock_Vina\ligantes\ligante.pdbqt → edite para ~\Dock_Vina\ligantes\*.pdbqt

Clique em CONVERT para converter todos os ligantes.

Vídeo: Conversão de múltiplos ligantes com Open Babel
6 Sexta etapa: Execução do Docking
1. Navegue até o diretório:
Linux → cd ~/Mult_lig
Windows → cd %USERPROFILE%\Mult_lig
2. Execute o comando:
python3 multiplo_lig.py
📄 Resultado

Será gerado o arquivo resultados_vina.csv delimitado por vírgulas com as energias dos ligantes.

Vídeo: Execução do docking para múltiplos ligantes
